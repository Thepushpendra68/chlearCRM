⚠️ IMPORTANT: Configure Your Environment Variables

The backend/.env file has been created but needs your actual values.

STEP-BY-STEP CONFIGURATION:

1. GEMINI_API_KEY (Required for AI features)
   ----------------------------------------
   a) Visit: https://makersuite.google.com/app/apikey
   b) Click "Create API Key" 
   c) Copy the key
   d) Open backend/.env
   e) Replace: GEMINI_API_KEY=your_gemini_api_key_here
      With:    GEMINI_API_KEY=AIzaSy...your_actual_key

2. SUPABASE Configuration (Required for database)
   ----------------------------------------------
   a) Get from your Supabase project dashboard
   b) Project Settings → API
   c) Copy "URL" and "service_role key"
   d) Update in backend/.env:
      SUPABASE_URL=https://xxxxx.supabase.co
      SUPABASE_SERVICE_KEY=eyJxxx...your_key

3. JWT_SECRET (Required for authentication)
   -----------------------------------------
   Generate a random string (32+ characters)
   Example: openssl rand -base64 32
   Or use: MySecureRandomString2024!@#$%^&*

4. Restart Backend
   ----------------
   cd backend
   npm run dev

VERIFY IT'S WORKING:
   - Look for: [EMAIL AI] AI service initialized successfully
   - Try AI Generate in Email Sequences

TROUBLESHOOTING:
   - If AI still fails, check the key is correct
   - Make sure there are no extra spaces
   - Restart backend after changes
